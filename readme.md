# block for text

### 구현된 사항 :
- 기초적인 블록코딩 시스템
- 변수의 스코프 단위 관리 -> 반복문, 함수 등 실행 시 적용 가능
- 기초적인 연산 및 입출력 연산자
- 자료형의 간단한 분리(Number, String)
- 블록 복사
- 값 블록과 일반 블록 구분
- 포인터 위치 기반 블록 드래그&드롭

### 미구현된 사항 :
- 순차 모드
- 변수값 표시
- 토큰 변경(한글 <-> 파이썬 지원 예정이었으나 현재 한글만 지원)
- cblock 형태만 갖추고 기능은 X
- 다크 모드
- 좌표 기반 블록 드래그&드롭 판정
